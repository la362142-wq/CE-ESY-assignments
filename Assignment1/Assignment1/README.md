# Assignment 1 – Circular Buffer

This assignment implements a simple circular buffer in C.

## Files
- **prog.c**: Contains the implementation of the circular buffer and the main function.

## How to Compile
Use the following command:
gcc prog.c -o prog

## How to Run
./prog

## Description
The program demonstrates:
- Initializing a circular buffer
- Adding elements
- Removing elements
- Checking if the buffer is full or empty
